# 註解 (Comments)

## 單行註解 `//`

```rs
fn main() {
    // 這是一個單行註解
    let x = 10; // 這也是註解，寫在程式碼後面
    println!("{x}");

    // let y = 10; ← 這行程式碼被註解掉了，所以不會被執行
}
```

## 多行註解 `/* ... */`

```rs
fn main() {
    /*
      這是第一行註解
      這是第二行註解
      這是第三行註解
    */
    let y = /* 這是嵌入行內的註解 */ 10;
    println!("{y}");
}
```

## 巢狀區塊註解 `/* /* ... */ ... */`

```rs
fn main() {
    /*
        /* 假設這行是原本的區塊註解 */
        let x = 10;
        println!("{x}"); // 假設這行是原本的單行註解
    */
    println!("只執行這行");
}
```

## 文件註解 `///`

````rs
/// 計算兩數之和
///
/// 範例：
///
/// ```rs
/// assert_eq!(add(1, 2), 3);
/// ```
fn add(a: i32, b: i32) -> i32 {
    a + b
}
````
